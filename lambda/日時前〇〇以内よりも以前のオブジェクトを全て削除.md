以下のように `removeIf` を使うのが一番簡単です。

## 例：`LocalDateTime` の日時フィールドで削除する

```java
import java.time.LocalDateTime;
import java.util.List;

public class Item {
    private LocalDateTime dateTime;

    public LocalDateTime getDateTime() {
        return dateTime;
    }
}
```

ある特定日時 `targetDateTime` の **1時間前よりも前** のデータを削除する場合です。

```java
LocalDateTime targetDateTime = LocalDateTime.of(2026, 6, 28, 10, 0);

LocalDateTime limit = targetDateTime.minusHours(1);

items.removeIf(item -> item.getDateTime().isBefore(limit));
```

例えば `targetDateTime` が `2026-06-28 10:00` の場合、

```text
limit = 2026-06-28 09:00
```

なので、削除されるのは次のようなデータです。

```text
08:59:59 → 削除
09:00:00 → 残る
09:30:00 → 残る
10:00:00 → 残る
```

つまり、**特定日時の1時間前より古いものだけ削除**します。

---

## メソッド化するとこうです

```java
public static void removeOlderThanOneHourBefore(
        List<Item> items,
        LocalDateTime targetDateTime
) {
    LocalDateTime limit = targetDateTime.minusHours(1);

    items.removeIf(item -> item.getDateTime().isBefore(limit));
}
```

使用例です。

```java
removeOlderThanOneHourBefore(items, LocalDateTime.now());
```

この場合は、**現在日時の1時間前より古いオブジェクト**が削除されます。

---

## null の可能性がある場合

`dateTime` が `null` の可能性があるなら、明示的に扱った方が安全です。

### null も削除する場合

```java
items.removeIf(item ->
        item.getDateTime() == null ||
        item.getDateTime().isBefore(limit)
);
```

### null は残す場合

```java
items.removeIf(item ->
        item.getDateTime() != null &&
        item.getDateTime().isBefore(limit)
);
```

---

## 元のリストを変更したくない場合

元のリストから削除するのではなく、条件に合うものだけ新しいリストにしたい場合です。

```java
List<Item> filteredItems = items.stream()
        .filter(item -> !item.getDateTime().isBefore(limit))
        .toList();
```

Java 8〜15 なら `toList()` ではなく次のように書きます。

```java
import java.util.stream.Collectors;

List<Item> filteredItems = items.stream()
        .filter(item -> !item.getDateTime().isBefore(limit))
        .collect(Collectors.toList());
```

---

基本形はこれです。

```java
LocalDateTime limit = targetDateTime.minusHours(1);
items.removeIf(item -> item.getDateTime().isBefore(limit));
```

`1時間` 固定ではなく、**規定分 + 60分** にするなら、`minusHours(1)` ではなく `minusMinutes(...)` を使います。

```java
int specifiedMinutes = 30; // 規定分
LocalDateTime targetDateTime = LocalDateTime.now();

LocalDateTime limit = targetDateTime.minusMinutes(specifiedMinutes + 60);

items.removeIf(item -> item.getDateTime().isBefore(limit));
```

例えば、規定分が `30分` の場合、

```text
規定分 + 60分 = 90分
```

なので、次と同じ意味になります。

```java
LocalDateTime limit = targetDateTime.minusMinutes(90);
```

つまり、`targetDateTime` が `10:00` なら、

```text
limit = 08:30
```

となり、削除されるのは次のようなデータです。

```text
08:29:59 → 削除
08:30:00 → 残る
09:00:00 → 残る
10:00:00 → 残る
```

メソッド化するとこうです。

```java
public static void removeOlderThanSpecifiedMinutesPlus60(
        List<Item> items,
        LocalDateTime targetDateTime,
        int specifiedMinutes
) {
    LocalDateTime limit = targetDateTime.minusMinutes(specifiedMinutes + 60L);

    items.removeIf(item -> item.getDateTime().isBefore(limit));
}
```

`specifiedMinutes + 60L` としているのは、`minusMinutes` の引数が `long` 型だからです。`int` のままでも動きますが、`60L` にしておくと型が明確です。

null を考慮するなら、例えば null も削除する場合はこうです。

```java
items.removeIf(item ->
        item.getDateTime() == null ||
        item.getDateTime().isBefore(limit)
);
```

基本形はこれです。

```java
LocalDateTime limit = targetDateTime.minusMinutes(specifiedMinutes + 60L);
items.removeIf(item -> item.getDateTime().isBefore(limit));
```
